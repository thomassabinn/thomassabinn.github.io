Sobre Mim

Olá! Meu nome é Thomas Sabin Almeida Xavier. Atualmente, estou cursando Análise e Desenvolvimento de Sistemas pela UNINTER, motivado pela minha paixão por tecnologia e pela vontade de transformar ideias em soluções práticas e eficientes. Desde jovem, sou fascinado por como a programação pode resolver problemas complexos e otimizar tarefas do dia a dia.

Meu objetivo é me tornar um desenvolvedor full-stack, com profundo conhecimento tanto no front-end quanto no back-end. Nas horas vagas, gosto de explorar novas tecnologias, editar vídeos, jogar jogos bobos como bloons td6, e me aprofundar em meus hobbies.

Portfólio de Projetos

Abaixo estão alguns projetos acadêmicos que desenvolvi para aplicar os conhecimentos adquiridos durante o curso.

Calculadora de IMC

Uma calculadora de Índice de Massa Corporal (IMC) desenvolvida com HTML, CSS e JavaScript básico. O projeto focou na manipulação de eventos e lógica de programação do lado do cliente.

Página de Login Estática

Criação de uma interface de login visualmente agradável, utilizando apenas HTML e CSS. O objetivo foi praticar a estruturação de formulários e a estilização avançada com Flexbox e Grid Layout.

Blog Simples

Desenvolvimento da estrutura de um blog simples com layout responsivo. O projeto serviu para aprofundar os conhecimentos em semântica HTML5 e técnicas de design que se adaptam a diferentes tamanhos de tela.
